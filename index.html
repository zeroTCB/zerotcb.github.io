About ZeroTCB

To put an end to data breaches and security exploits, we are building state-of-the-art solutions that allow customers to perform sensitive data analytics in both public and private clouds. Our mantra is simple: remove large parts of the cloud software infrastructure (such as the cloud OS, hypervisors, etc.) from the trusted computing base, and instead rely only on trusted hardware primitives found in modern platforms such as Intel SGX. As a result, neither the attackers that compromise the cloud platform software, nor a rogue administrator with root access can access the customer's data. We ensure that your sensitive data and its cryptographic keys are never available in these untrusted platform layers --- your data is always protected.

We are currently working on securing data analytics computation (e.g. MapReduce, Spark jobs). Our framework allows a customer to offload both code and data in an encrypted form to the public cloud, perform the computation in a secure way, and return the encrypted result back to the user.
